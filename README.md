# OnyxCloud Game Server Hosting Platform

Monorepo skeleton for a production-ready game server hosting platform powered by the Pterodactyl Panel API.

## Stack
- Backend: Node.js + Express + JWT auth
- Frontend: Next.js + Tailwind CSS
- Database: MySQL
- Architecture: REST API with layered backend modules

## Core modules covered by this structure
1. User registration/login
2. Wallet billing and ledger logging
3. UPI QR deposits with admin approval workflow
4. Game server deployment through Pterodactyl API
5. Admin panel domains (deposits, users, servers)

See `PROJECT_STRUCTURE.md` for the complete folder and file map.
